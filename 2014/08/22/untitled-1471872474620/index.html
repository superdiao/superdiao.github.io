使用Hexo搭建个人博客

前言

之前有用过别的博客写过，不过有的要审核有的界面不喜欢，也没有什么特色，总是提不起兴趣，后来接触了github，感觉到了写博客的动力，就自己搭建了一个个人博客。因为这中间遇到了很多问题，所以特此编写了一个教程，方便初学者使用，少走弯路。

正文

此教程适用于windows，不适用mac和linux，之后有可能会写linux的教程，在此不做确定。

安装Node.js

下载安装Node.js。安装完之后建议重启下。如果不重启后面使用hexo的时候有可能会遇提示无效的命令。

安装Git

下载并安装msysgit。建议从官网下载，如果慢的话建议用迅雷下载，网速好应该会很快。
下载地址：https://git-for-windows.github.io/

安装hexo

安装之前介绍几个hexo常用命令，#后面是注释。这几个在之后的使用中会经常用到，建议先学习一下

  1. $ hexo g #完整命令为hexo generate,用于生成静态文件
  2. $ hexo s #完整命令为hexo server,用于启动服务器，主要用来本地预览
  3. $ hexo d #完整命令为hexo deploy,用于将本地文件发布到github上
  4. $ hexo n #完整命令为hexo new,用于新建一篇文章

装完git后在桌面任意地方鼠标点击右键会看到有Git Bash和Git GUI。在任意地方鼠标右键后点击Git Bash即可弹出终端，在终端中使用下面命令安装

  1. $ npm install hexo-cli -g

如果之后在使用的过程中，遇到以下的错误

  1. ERROR Deployer not found : github

则运行以下命令,或者你直接先运行这个命令更好。如果不运行之后还可能会报错。

  1. $ npm install hexo-deployer-git –save
  2. $ hexo g
  3. $ hexo d

接下来创建放置博客文件的地方：hexo文件夹。创建在自己习惯使用的地方，名字可以随意起，只要自己记得住就好。我方便记直接使用了hexo。
比如我的文件夹位置为：c:\hexo。创建文件夹之后进入该文件夹，右键选择Git Bash进入终端，执行以下命令：

  1. $ hexo init

安装依赖包

  1. $ npm install

在E:\hexo内执行以下命令：

  1. $ hexo g
  2. $ hexo s

然后用浏览器访问http://localhost:4000/，这个时候你应该可以看到一个博客了，不过这个博客是自带的，存在本地，别人是看不到的。hexo3.0使用的默认主题是landscape。这也正是hexo的强大之处，可以本地预览，之后再上传到github上即可。

注册Github

上github注册账号
https://github.com/

创建repository

repository相当于一个仓库，用来存放你的代码。而且可以提取很久之前修改过的代码，非常强大，而且是免费。这也正是github 的强大之处。首先登陆github，进入个人页面。点击New repository新建一个仓库。如果没有看到New repository 的话进入repository后即可看到。

创建时，只需要填写Repository name即可，当然这个名字的格式必须为yourname.github.io，例如我的为superdiao.github.io

部署本地文件到github

repository创建后，就是要把博客放到Github上了，编辑C:\hexo文件下的_config.yml文件。建议使用代码编辑器打开。

在_config.yml最下方，添加如下配置(命令中的第一个superdiao为Github的用户名,第二个superdiao为之前New的Repository的名字,记得改成自己的。另外记得一点，hexo的配置文件中任何’:’后面都是带一个空格的),如果配置以下命令出现ERROR Deployer not found : github，则参考上文的解决方法。

  1. deploy:
  2. type: git
  3. repository: http://github.com/superdiao/superdiao.github.io.git
  4. branch: master

配置好_config.yml并保存后，执行以下命令部署到Github上。

  1. $ hexo g
  2. $ hexo d

执行上面的第二个命令，可能会要你输入用户名和密码，输入github的帐号密码即可。
如果_config.yml中的repository: http://github.com/superdiao/superdiao.github.io.git可以打开的话，在浏览器中访问superdiao.github.io 就能看到自己的博客了。
